$\color{green}{\text{P1982}}$

人生苦短，我用 python

按照题意求一下前缀最大子段和，然后模拟一下。

但是会爆 long long，于是我们用 python 高精就能解决这道题目。

$\color{green}{\text{P1043}}$

简单区间 dp。

断环成链，设 $f_{i,j,k}$ 表示从第 $i$ 位到第 $j$ 位，分 $k$ 段得到的最大值。

然后设 $g_{i,j,k}$ 表示从第 $i$ 位到第 $j$ 位，分 $k$ 段得到的最小值。

随便转移转移就好了。
