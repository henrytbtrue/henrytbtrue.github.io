NOIP2021游记

发表于 2021-11-21 更新于 2021-12-05 分类于 游记 阅读次数： Valine： 本文字数： 2.2k 阅读时长 ≈ 2 分钟

Day -?

以约 12.7×104\frac{1}{2.7\times 10^4}2.7×1041​ 的概率在 B 站抽中了一个价值 988988988 元的手办。

RP–（记住这要考）。

Day 0

疯狂背板子（最后一个都没有用上）。

带了一点饮料和水，没带吃的。

Day 1

去了八十中，回想了去年在这里撞大运卡过 1= 线的经历，觉得今年不会这样了（flag）。

开考前监考老师不停提醒开始考试前不要写任何代码、最好不要使用 windows 系统（重点）。我配置了下 vscode，感觉电脑有点卡（此时还没有意识到问题的严重性）。

解压密码 IronHeart@413A？？盲猜是个游戏名。

开始考试。过了眼题面。

  • T1 第一眼口胡了个长得像欧拉筛的诡异筛法，个人感觉过不了。
  • T2 不会，不过一眼看出来一定是个 dp。
  • T3 目前不会，过会可以推推性质。
  • T4 肯定不会，但是暴力分非常简单而且有 323232 分的巨额分数。

写了写 T1。写的时候电脑不停地卡顿，不过好在等个十几秒就能接着写。

写完了，花了半个小时左右。

编译编译。

虚拟机卡死了。

等了十几秒，没有好。

等了五分钟，没有好。

等了十分钟，没有好。

这时我意识到了这次卡顿的严重性（吗？），于是使用各种方法尝试强制重启虚拟机，又过了 5 分钟终于重启了。

重启了，经过了一些卡顿之后终于编译好了，然后测样例，然后测大样例，发现这个诡异筛法大样例跑得飞快，就扔了 T1。

打算登上程序回收系统交题。

？？？我虚拟机重启后网络连不上程序回收系统了？？？（出考场发现别的重启虚拟机的同学都没有这个问题）

然后举手叫监考老师（现在看来是极其错误的决定，当时要是直接出去用 windows 交就好了，可惜我还是秉承了不要用 windows 的原则）。监考老师认为我的“重启方式不对”，让我再按照他说的方法重启一次。

重启了一次，没用。

重启了两次，没用。监考老师还是不死心。

重启了三次，没用。监考老师只好让我先接着做，等他叫另一个老师过来。

这个插曲严重打乱了我的想题思路，后面也没想出什么本质性/突破性的东西了。

电脑之后一直是死机-重启-过一会继续死机的循环。

扔了 T2（打算过会写暴力），然后开始搞 T3。

在纸上推了推式子，写了一些情况，一开始没发现什么，只搞出了 n=4n=4n=4 的做法。

后来又观察了下题面，发现了开局 aia_iai​ 是单调不降的，我想到了差分。

于是又写了些情况，发现进行操作后，差分数组只是除了第一位其它都被打乱了顺序而已，元素本身是不变的。然后 n!n!n! 暴力诞生了。

这时候，来了一位老师，他对我的虚拟机进行了很多次重启，对网络配置进行了一通操作，但是没有任何用处。过了十几分钟后他也死心了，让我看看能不能复制出 windows 系统交题，结果发现可以。早知道最开始不要找老师就好了，谁叫老师一开始不让用 windows。

我还觉得可以接着想 T3，于是尝试找差分数组的分配顺序有没有规律。

伴随卡顿与几次重启输出了一些中间变量，手造了一些样例，发现一定是一段递减然后一段递增。

于是 nlog⁡n⋅2nn\log n\cdot 2^nnlogn⋅2n 的暴力出现，具体就是枚举每个元素在递减段还是递增段出现。

然后就没看出什么性质，在极其卡顿的机器上写完了这个暴力。期望得分 484848 分。

这时来了个老师问我要不要延时，但是最多能延 3∼53\sim 53∼5 分钟（理由是电脑卡顿、老师修电脑的时候我还可以想题，但实际上严重打乱了我的思维，对我的影响远超过 3∼53\sim 53∼5 分钟）。我只好跟她说能延多少是多少吧。

写了 T2 暴力，没过大样例，调了一小会，重启了一次，结果发现时间就只剩下半个小时，不够写（很悬）T4 暴力了。心态逐渐爆炸。

时间就是这么快。

这时候又来了个老师，跟我说“特派员说北京最好不要延时，你这种最多延 555 分钟造成不了什么大影响，所以还是算了吧。”然后我延时就没了。

最后 202020 分钟是折磨的 202020 分钟，我明明知道 T4 暴力摆在那里，多么好拿，但就是不能写，因为写了也写不完，在策略上不如去调 T2 暴力，最后也没调出来。

彻底去世了。

估分 100+(0∼30)+48+0100+(0\sim 30)+48+0100+(0∼30)+48+0。一等随缘。

考试结束之后，被这个电脑折磨透顶了的我充满这满腔的怒火，向旁边的同校同学使劲抱怨。同校同学们有的没被电脑坑，有的也被坑了但远远没我惨。回忆了一下，当时整个考场从始至终都是我举手找监考老师的声音。

初三了，接下来好好搞文化课吧，别连高中都考不上了。

出考场，发了个 pyq。

本次的体验好比在中考考场上场场考试都流鼻血滴到答题纸上导致在写完一半题的情况下换答题纸。

并且写一句话就要换根笔，答题纸翻个面就要花两分钟（from sqrt_7）。

自闭了。

出来看估分。

LG：100+5+48+0=153100+5+48+0=153100+5+48+0=153。

小图灵：100+25+48+0=173100+25+48+0=173100+25+48+0=173。

infoj：100+0+48+0=148100+0+48+0=148100+0+48+0=148。

死得透透。感谢八十中的电脑。

Day ?
